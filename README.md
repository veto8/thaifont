Welcome to ThaiFont! This project provides a unique font designed to help you learn to read Thai script by highlighting the vocal sounds within the words.  It's a visual aid to bridge the gap between spoken and written Thai.

## The Idea Behind ThaiFont

Learning to read Thai can be challenging.  Thai script doesn't always explicitly represent all the vowel sounds, and the tone marks can seem daunting. ThaiFont aims to make the process easier by:

*   **Visually Highlighting Vowels:** The font subtly emphasizes the vowel sounds in each word, making them easier to identify.
*   **Improving Comprehension:** By connecting the written script with the spoken sounds, you can improve your reading comprehension and pronunciation.
*   **Making Learning Fun:**  We believe that learning should be enjoyable! ThaiFont offers a fresh and engaging way to approach Thai script.

## How to Use ThaiFont

1.  **Download the Font:** Grab the latest version of the font from the [Releases](https://github.com/myridia/thaifont/releases) section.
2.  **Install the Font:** Install the font on your computer (usually by double-clicking the font file and following the instructions).
3.  **Use in Your Favorite Text Editor or Application:** Select ThaiFont as the font in your word processor, text editor, or any application that supports custom fonts.
4.  **Start Reading!**  Open a Thai text document and start reading. Pay attention to the highlighted vowels and how they relate to the pronunciation.
